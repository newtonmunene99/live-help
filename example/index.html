• Live Help Component

SIMPLE, CUSTOMIZABLE, FREE, OPEN SOURCE

Our Live Help Component is easily customizable using css. It comes with a free Dashboard built in a minimal UI.

Use One Of the following methods

  • NPM

    • In Your Project run npm install live-help

  • Script Tag

    • Add this script at the bottom of your page

Then you can place live-help anywhere

Live Help Takes a few attributes that are not mandatory

  • Organization

    • This shows your organization's name. If not supplied, it defaults to 'Live Help'

  • Color

    • This is the theme color to be used. It can either be a color name or a hex color code. If not supplied it defaults to 'Blue'
